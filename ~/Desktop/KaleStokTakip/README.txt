KALE STOK TAKİP UYGULAMASI
=======================

Kurulum ve Çalıştırma Talimatları:

1. Sistem Gereksinimleri:
   - Windows 10 veya Windows 11 (64-bit)
   - İnternet bağlantısı
   - Modern bir web tarayıcı (Chrome, Firefox veya Edge önerilir)

2. Uygulamayı çalıştırmak için:
   - start.bat dosyasını çift tıklayın
   - Uygulama otomatik olarak başlayacak ve tarayıcınızda açılacaktır

3. İlk çalıştırmada Windows Güvenlik Duvarı uyarısı çıkabilir:
   - "Erişime İzin Ver" seçeneğini tıklayın
   - Bu, uygulamanın yerel ağda çalışabilmesi için gereklidir

4. Uygulamayı kapatmak için:
   - Tarayıcı penceresini kapatın
   - Görev çubuğundaki SalesAutomationAPI simgesine sağ tıklayıp "Çıkış" seçeneğini seçin

Not: 
- Uygulama 7294 portunu kullanmaktadır. Bu portun başka bir uygulama tarafından kullanılmadığından emin olun.
- Veritabanı bağlantısı için internet bağlantınızın olması gerekmektedir.
- Uygulama ilk açılışta biraz yavaş başlayabilir, lütfen bekleyin.

Sorun Giderme:
- Eğer uygulama açılmazsa veya "Bu siteye ulaşılamıyor" hatası alırsanız:
  1. Windows Güvenlik Duvarı ayarlarını kontrol edin
  2. Antivirüs programınızın uygulamayı engellemediğinden emin olun
  3. 7294 portunu kullanan başka bir uygulama olmadığından emin olun
  4. Uygulamayı yönetici olarak çalıştırmayı deneyin
